Welcome to your new Jekyll blog! This is your first blog post, created to demonstrate how Jekyll works with the Minima theme.

## Getting Started with Jekyll

Jekyll is a static site generator that's perfect for blogs and simple websites. Here's what makes it great:

- **Simple**: Write your posts in Markdown
- **Fast**: Static sites load quickly
- **Flexible**: Customize with themes and plugins
- **Free**: Host on GitHub Pages for free

## Code Highlighting

Jekyll comes with syntax highlighting built-in. Here's an example:

```ruby
def hello_world
  puts "Hello, Jekyll world!"
end

hello_world
```

## What's Next?

You can now:

1. Edit this post in `_posts/2024-09-14-welcome-to-jekyll.markdown`
2. Create new posts in the `_posts` directory
3. Customize your site by editing `_config.yml`
4. Add pages by creating new `.md` files

Check out the [Jekyll docs][jekyll-docs] for more info on how to get the most out of Jekyll. File all bugs/feature requests at [Jekyll's GitHub repo][jekyll-gh].
